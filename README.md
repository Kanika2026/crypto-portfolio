# 🚀 Crypto Portfolio Tracker

A modern and responsive **Crypto Portfolio Tracker** built with **React + Vite**. This application enables users to explore real-time cryptocurrency market data, search coins instantly, analyze detailed coin information, manage a personal portfolio, and maintain a watchlist.

---

## 📌 Overview

Crypto Portfolio Tracker is a cryptocurrency dashboard that integrates with a public crypto API to provide live market information. The application focuses on clean UI, responsive design, fast performance, and an intuitive user experience.

---

## ✨ Features

- 📊 View Top 50 Cryptocurrencies
- 🔍 Debounced Search Functionality
- 📈 Detailed Coin Information
- 💰 Portfolio Management
- ⭐ Add & Remove Watchlist
- 📉 Interactive Price Charts
- 🌙 Dark Mode Support
- 📱 Fully Responsive Design
- ⚡ Fast Performance with Vite
- 🔄 Automatic Data Refresh
- 🎨 Professional Dashboard UI

## ⚙️ Installation & Setup

### Clone Repository

```bash
git clone https://github.com/Kanika2026/crypto-portfolio.git
```

### Navigate to Project

```bash
cd crypto-portfolio
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open:

```
http://localhost:5173
```

### Production Build

```bash
npm run build
```
